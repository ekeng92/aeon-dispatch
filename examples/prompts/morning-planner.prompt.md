# Morning Planner

Review my current work context and help me plan the day.

## Steps

1. **Check git status** across repos in this workspace: uncommitted changes, branches, recent commits
2. **Scan for open issues or tasks** if a task board, TODO file, or GitHub Issues are available
3. **Identify priorities**: what's in progress, what's blocked, what's due soon
4. **Suggest a plan** for today: 3-5 concrete tasks in priority order, with estimated effort (small/medium/large)

## Output Format

```
## Today's Plan — [date]

### Carry-over (in progress)
- [ ] ...

### Priority tasks
1. ...
2. ...
3. ...

### If time permits
- ...

### Blockers
- ...
```

Keep it concise. I want to scan this in 30 seconds and start working.
